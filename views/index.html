API Project: Timestamp Microservice

Timestamp Microservice API is made to take user input in date and time format in the url and then display date and time in JSON.

Example Usage:

  • url/api/timestamp/2015-12-25
  • url/api/timestamp/1451001600000

Example Output:

{"unix":1451001600000,"utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
